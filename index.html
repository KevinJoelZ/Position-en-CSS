Posición Static

La posición estática es el valor por defecto. Los elementos se posicionan según el flujo normal del documento.

La posición estática es el valor por defecto que tienen todos los elementos HTML. Cuando un elemento tiene la posición estática, se coloca en el flujo normal del documento, según el orden en que aparece en el HTML. Esto significa que los elementos se posicionan uno detrás de otro, de arriba a abajo y de izquierda a derecha, según el flujo de bloques o en línea.

Los elementos con posición estática no se ven afectados por las propiedades de desplazamiento top, right, bottom y left. Estas propiedades no tienen ningún efecto en un elemento estático, ya que se basa en la disposición natural del flujo del documento.

Posición Relative

La posición relativa se basa en la posición estática del elemento, pero se puede ajustar usando las propiedades top, right, bottom y left.

La posición relativa permite desplazar un elemento respecto a su posición original en el flujo del documento. Se puede utilizar las propiedades top, right, bottom y left para ajustar su posición con respecto a sí mismo.

Es importante destacar que un elemento con posición relativa no se quita del flujo normal del documento, por lo que el espacio que ocupa en el diseño aún está reservado para él.

Posición Absolute

La posición absoluta se posiciona respecto al primer elemento ancestro que no tenga la posición estática. Si no hay tal elemento, se posiciona respecto al elemento raíz.

La posición absoluta posiciona un elemento en relación con su primer elemento padre que tenga una posición diferente de estática (es decir, relativa, absoluta o fija). Si no hay un padre con posición establecida, se posiciona en relación con el elemento raíz (normalmente el cuerpo del documento).

Los elementos con posición absoluta se desplazan fuera del flujo normal del documento, lo que significa que no ocupan espacio en el diseño y pueden superponerse a otros elementos.

Posición Fixed

La posición fija fija el elemento respecto a la ventana del navegador. No se desplaza cuando se hace scroll.

La posición fija fija el elemento en una posición específica con respecto a la ventana del navegador. Este tipo de posición es útil para elementos que deben permanecer visibles mientras el usuario se desplaza por la página, como barras de navegación o anuncios.

Los elementos con posición fija no se mueven cuando se hace scroll, manteniéndose en la misma posición relativa al borde de la ventana del navegador.

Posición Sticky

La posición sticky alterna entre relativa y fija, dependiendo de la posición del scroll. Es útil para menús o elementos que deben estar siempre visibles.

La posición sticky combina características de las posiciones relativa y fija. Se comporta como posición relativa hasta que un elemento alcanza una posición específica durante el scroll, momento en el cual se fija en esa posición. Esto es útil para elementos que deben ser visibles mientras el usuario se desplaza por la página, pero que luego deberían comportarse como elementos relativos.

Los elementos con posición sticky se quedan "pegados" a una ubicación específica en la ventana del navegador durante el scroll, pero continúan respetando el flujo normal del documento hasta que alcanzan su punto de fijación.
